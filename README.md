# 🎮 Controls

| Control | Action |
|----------|--------|
| 🖱 Mouse | Steer Missile |
| W / S | Pitch Up / Down |
| A / D | Turn Left / Right |
| Arrow Keys | Alternative Flight Controls |
| Left Shift | Boost |
| ESC | Release Mouse |

---

# 🎯 Mission

Your objective is simple:

- 🚀 Launch the missile
- 🏛 Navigate the obstacle course
- 🏠 Fly through the building opening
- 🚇 Pass through the tunnel
- 🎯 Destroy the target
- 💥 Avoid crashing
